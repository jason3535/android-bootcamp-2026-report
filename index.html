AICore 非 LLM 模型支持

OEM 可在不满足 LLM 要求的设备上集成 AICore，最低仅需 6GB RAM：

Three Model Options

  • 语义搜索：300M 参数嵌入模型，自然语言查找
  • 轻量级 ASR & TTS：270M 参数 Transformer
  • 主动辅助：20-50M 参数动作模型

Impact for Nothing

  • 中端设备（6GB+）可接入 AI 能力
  • 预装 AICore APK 即可启用
  • 不依赖 Gemini Nano，降低硬件门槛

重点 AI 功能

My Voice Live Translate API

  • 实时语音翻译，保留说话者原始声音
  • 支持电话和视频通话，附带翻译字幕
  • 发布时间：2026 Q2

Notification Summary

  • 智能总结通知和长对话，支持 WhatsApp / Google Messages
  • 三层级展示：L0 截断(0 tap) → L1 完整摘要(1 tap) → L2 全文(2 taps)
  • 90%+ 质量通过率，0% 安全违规率
  • 语言路线：EN → JA/DE/KO (Q1) → ES/FR/IT/HI/ZH (Q3)
  • 需要：SystemUI + AiCore (Nano V2+) + PCC (OEM 版 Q2 分发)

Scam Detection

  • 检测 SMS/MMS/RCS 对话诈骗，本地处理保护隐私 Available

AI VM 架构 (Long-term)

  • Protected VM (pVM) 构建安全个性化 AI 体验
  • 敏感用户数据在 VM 内创建/存储/访问
  • OEM 可定制：应用、数据捐赠、Use case delegate
  • 数据流：Android→VM (捐赠) → VM→VM (处理) → VM→Android (AI 体验)
